# User Stories: Employer

**Test account:** `employer@demo.elasticos.com` / `Demo123!`

---

## Authentication & Access

- **US-E01** — As an Employer, I want to log in with my email and password so that I can access my ElasticOS dashboard.
- **US-E02** — As an Employer, I want to sign out so that my session is securely ended.
- **US-E03** — As an Employer, I want to see my role (Employer) and Employer ID on the dashboard so that I know I am in the right account.

---

## Workforce Roster

- **US-E04** — As an Employer, I want to view my workforce roster so that I can see all workers affiliated with my organisation.
- **US-E05** — As an Employer, I want to import workers from a CSV file (workerId, engagement) so that I can bulk-add workers to my roster.
- **US-E06** — As an Employer, I want to download a CSV template for roster import so that I know the correct format.
- **US-E07** — As an Employer, I want to export my roster as CSV so that I can back up or analyse my workforce data.
- **US-E08** — As an Employer, I want to see each worker’s email, department, role, and contract type in the roster so that I can manage my organisation structure.

---

## Employment Affiliation Ledger

- **US-E09** — As an Employer, I want to create affiliation records (ACTIVE, ADJUSTED, TERMINATED, REACTIVATED) for workers so that the Employment Affiliation Ledger reflects engagement accurately.
- **US-E10** — As an Employer, I want to set engagement intensity (0–1) when adding or adjusting workers so that I can model part-time or scaled engagement.
- **US-E11** — As an Employer, I want to view my organisation’s affiliation history via the API so that I can audit and report on workforce engagement.

---

## Organisation Structure

- **US-E12** — As an Employer, I want to create departments (with optional parent) so that I can organise my workforce hierarchically.
- **US-E13** — As an Employer, I want to create job roles (with optional department) so that I can define positions in my organisation.

---

## Verification & Compliance

- **US-E14** — As an Employer, I want to verify my legal entity via Companies House so that I can prove my organisation is legitimate.
- **US-E15** — As an Employer, I want verification attempts logged so that there is an audit trail for compliance.

---

## App Marketplace

- **US-E16** — As an Employer, I want to access the App Marketplace from my dashboard so that I can discover and use workflow, reporting, and compliance tools.

---

## Navigation

- **US-E17** — As an Employer, I want to navigate between Dashboard and Workforce Roster so that I can access all my features easily.
