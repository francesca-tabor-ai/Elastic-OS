# User Stories: Worker

**Test account:** `worker@demo.elasticos.com` / `Demo123!`

---

## Authentication & Access

- **US-W01** — As a Worker, I want to log in with my email and password so that I can access my ElasticOS dashboard.
- **US-W02** — As a Worker, I want to sign out so that my session is securely ended.
- **US-W03** — As a Worker, I want to see my role (Worker) and Worker ID on the dashboard so that I know I am in the right account.

---

## Worker Profile

- **US-W04** — As a Worker, I want to view my profile (headline, bio) so that I can see how I present myself to employers.
- **US-W05** — As a Worker, I want to edit my headline and bio so that I can keep my profile up to date.
- **US-W06** — As a Worker, I want to add skills with a proficiency level (1–5) so that employers can match me to opportunities.
- **US-W07** — As a Worker, I want to add certifications (name, issuer, dates, document URL) so that I can showcase my qualifications.
- **US-W08** — As a Worker, I want to add portfolio items (title, description, links) so that I can demonstrate my work.
- **US-W09** — As a Worker, I want to view employer references on my profile so that I can see feedback from past employers.

---

## Employment History

- **US-W10** — As a Worker, I want to view my affiliation history so that I can see my employment timeline across employers.
- **US-W11** — As a Worker, I want to see engagement intensity and record type (ACTIVE, TERMINATED, etc.) for each affiliation so that I understand my employment status.

---

## App Marketplace

- **US-W12** — As a Worker, I want to access the App Marketplace from my dashboard so that I can discover and use productivity tools.

---

## Verification & Security

- **US-W13** — As a Worker, I want to verify my government ID so that I can prove my identity to employers.
- **US-W14** — As a Worker, I want to set up MFA (multi-factor authentication) so that my account is more secure.

---

## Navigation

- **US-W15** — As a Worker, I want to navigate between Dashboard, Worker Profile, Employment History, and Marketplace so that I can access all my features easily.
